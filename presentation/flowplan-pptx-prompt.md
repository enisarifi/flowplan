## PROMPT START

Create a professional, modern PowerPoint presentation (16:9 aspect ratio) for a college Entrepreneurship class group presentation. The product is called **FlowPlan** — an AI-powered study planner.

### Design Guidelines
- Clean, minimal, modern SaaS-style design
- Primary brand color: Indigo (#6366F1) with white backgrounds
- Secondary accents: Slate gray for body text, light indigo (#EEF2FF) for card backgrounds
- Font: Use a clean sans-serif (Inter, Poppins, or similar)
- Use icons where possible (brain, calendar, chart, lightning bolt, target, rocket)
- Avoid clip art — keep it professional and startup-like
- Each slide should have minimal text — use bullet points, not paragraphs
- Include subtle visual elements (gradients, rounded cards, thin dividers)

### SLIDE 4 — The Solution

### SLIDE 5 — How It Works
**Title:** How It Works
**Content:** 4-step horizontal flow diagram with icons and arrows:

3. **AI Generates Schedule** — Personalized 7-day plan placed at your optimal times
4. **Study, Feedback, Adapt** — Complete sessions → rate them → AI adapts next week

**Visual:** Horizontal numbered steps (1→2→3→4) with icons above each step. Clean process diagram style.
